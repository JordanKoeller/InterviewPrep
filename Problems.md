### Data Structures

Implement the following datastructures

### Algorithms

Be ready to pull out the following algorithms

  1. Quicksort
  2. Mergesort
  4. Breadth-first/Depth-first searches
  5. Topological sort
  6. Dijkstra's
  7. Binary Search
